************
Installation
************

Requirements
============

- Python 3.*
- pip 20.0.2

Installing by using PyPi
========================

Install
-------
.. code-block:: bash

	git clone https://github.com/intsystems/2023-Problem-140.git /tmp/ProjectTemplate
	python3 -m pip install /tmp/ProjectTemplate/src/

Uninstall
---------
.. code-block:: bash

  python3 -m pip uninstall mylib
